Trois approches pour le diagramme en arête de poisson

Comment rendre la structure fishbone pour l'Ishikawa ? Clique sur une approche pour la sélectionner.

A — SVG pur

Les arêtes et la colonne vertébrale sont dessinées en SVG. Les zones de dépôt sont des éléments HTML positionnés par-dessus.

+

  • Rendu fidèle à l'original
  • Arêtes vraiment diagonales

−

  • Drag-and-drop en SVG plus complexe
  • Responsive difficile

B — CSS + transforms

Les arêtes sont des <div> avec transform: rotate(±40deg). Les zones de dépôt restent des éléments HTML standards.

SVG uniquement pour dessiner les lignes diagonales (décor). Les zones de dépôt et cartes sont des div HTML positionnés par-dessus.
